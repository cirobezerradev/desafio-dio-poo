# Desafio POO - Bootcamp Luizalabs DIO

### Objetivo
> Empregar todo aprendizado de POO, criando as classes conforme UML apresentado pelo professor para fazermos a atualização do sistema bancário do [desafio Anterior sobre funções](https://github.com/cirobezerradev/desafio-dio-01-funcoes).

Na entrega do desafio criei todas as classe conforme estabelecido no desafio, e da parte do desafio extra,deixei pronta as funcionalidades de Cadastro de Cliente, Listagem de Cliente, Cadastro de Conta e Listagem de Contas Cadastradas. E Irei acrescentar as demais funcionalidades no decorrer do bootcamp.

### Conclusão:
No desafio principal de criar as classes pude empregar os conceitos de classe, construtores, classes abstratas, properties, métodos abstratos, metodos de classe, herança, encapsulamento e polimorfismo.
